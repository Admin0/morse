FEATURES

• LETTERS ⇄ MORSE - Translate to Morse code from English or etc. Of course reverse are possible. • REAL TIME TRANSLATE - Why do you need to click every time [TRANSLATE] button for? Finally freed from the painful task. • Somehow cool. • Don't know why, but hot. • Unknown, but attractive. • LETTERS ⇄ BRAILLE - Translate to Braille from English or etc. 　

HOW TO USE

• Type letters in the input box to translate. • Tap the icon at right side of input box. after that, type code in the input box to analyze. 　 ------------------------------------- Translations between the following languages are supported: Latin(English, etc.), Arabic(Arabic, Persian, etc.), Cyrillic(Russian, etc.), Greek(Greek), Hangul(Korean), Hebrew(Hebrew or etc.), Kana(Japanese)
